JavaScript：sort()排序不正确
2020-07-29
在JavaScript中，数组使用sort()后发现有排序不正确的情况排序前var a = [10,6,6,10,10,9,8,8,3,3,8,2,1,5,1,9,5,2,7,4,7,7]排序后很明显排序不正确查阅资料后，数组元素首先会被转换为字符，例如1变成"1",之后会根据Unicode编码的顺序来进行排序，所以这就是为什么10排到了2前面解决方法是，指定好compareFunction 12345678910a.sort(function (a,b) { if (a < b ) { return -1; } if (a > b ) { return 1; } // a must be equal to b return 0; }); 这样排序后就没有问题了
