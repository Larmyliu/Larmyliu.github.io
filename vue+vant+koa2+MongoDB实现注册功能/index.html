    Schema，相当于MongoDB数据库的一个映射
    • Schema中的数据类型
    • String ：字符串类型
    • Number ：数字类
    • Date ： 日期类型
    • Boolean： 布尔类型
    • Buffer ： NodeJS buffer 类型
    • ObjectID ： 主键,一种特殊而且非常重要的类型
    • Mixed ：混合类型
    • Array ：集合类型
