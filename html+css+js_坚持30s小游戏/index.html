坚持30s小游戏


用到：

  1. 弹性运动
  2. 拖拽事件
  3. 碰撞检测
  4. 定时器清除
  5. 单对象编程

弹性运动：
在绿色小球碰到四个壁时要方向要发生改变，运动方向变成原来的-1
拖拽事件：鼠标点下去时的点，和移动过程中产生的新点之间的差值就是移动的距离，移动过程中要不断更新上一个点的值，保证持续产生差值；
碰撞检测：
当两个点的中心距离小于两个球的半径之和即为碰撞
最后游戏结束后要把所有的定时器给清除
